### Теорема о связи функции, имеющей конечный предел, с ее пределом и б.м.ф.

> Пусть функция $\large f(x)$ определена в проколотой окрестности точки $\large x_0$. Для того, чтобы $\large f(x)$ имела в точке $\large x_0$ предел равный $\large A$, *необходимо и достаточно*, чтобы $\large f(x)$ можно было представить в виде суммы $$\Large f(x) = A+\alpha(x)$$ где $\large \alpha(x)$ - б.м.ф. при $\large x\to x_0$

##### Доказательство
$\square$
###### Необходимость

Пусть $\large \displaystyle\lim_{x\to x_0}f(x) = A$

Положим $\large \alpha(x) =f(x) - A$ и докажем, что $\large \alpha(x)$ - б.м.ф. при $\large x \to x_0$

Возьмем любое $\large \varepsilon >0$

Т.к.  $\large \displaystyle\lim_{x\to x_0}f(x) = A$, то для выбранного $\large \varepsilon$ найдется такое $\large \delta > 0$, что для всех $\large x : 0 < |x-x_0| <\delta$ верно неравенство
$$\Large |f(x) - A| < \varepsilon$$

Поскольку мы выбрали  $\large \alpha(x) =f(x) - A$, то справедливо
$$\Large|\alpha(x)|<\varepsilon$$
Это означает, что $\large \alpha(x)$ - б.м.ф. при $\large x\to x_0$
###### Достаточность
Пусть функцию $\large f(x)$ можно представить в виде
$$\Large f(x) = A+\alpha(x)$$
где $\large A$ - постоянная, а $\large \alpha(x)$ - б.м.ф. при $\large x\to x_0$

Докажем, что $\large f(x)$ в точке $\large x_0$ имеет предел равный $\large A$

Возьмем любое $\large \varepsilon > 0$

Т.к. $\large \alpha(x)$ - б.м.ф при $\large x\to x_0$, то для выбранного $\large \varepsilon$ найдется такое $\large \delta > 0$, что для всех $\large x : 0 < |x-x_0|<\delta$ будет верно
$$\Large |\alpha(x)| < \varepsilon$$

Поскольку $\large f(x) = A + \alpha(x) \Rightarrow \alpha(x) = f(x)-A$, для тех же $\large x$ справедливо
$$\Large |f(x) - A| < \varepsilon$$

То есть $\large \displaystyle\lim_{x\to x_0}f(x) = A$ 

$\blacksquare$
