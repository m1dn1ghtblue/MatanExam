## Формула Тейлора для ФНП
Пусть функция $\large z = f(x,y)$ имеет непрерывные частные производные до $\large n$-го порядка включительно во всех точках $\large (x,y)$ некоторой $\large \delta$-окрестности точки $\large (x_0,y_0)$ и пусть точка $\large (x_0+\Delta x, y_0 + \Delta y)$ принадлежит этой окрестности

Положим
$$\Large x = x_0+t\Delta x,~~y = y_0+t\Delta y$$
где $\large t \in [0,~1]$ - новая независимая переменная. Тогда 
$$\Large z = f(x,y) = f(x_0+t\Delta x,~y_0+t\Delta y) = \varphi(t)$$
так что величина $\large z$ оказывается сложной функцией от $\large t$, определенной на отрезке $\large [0,~1]$ и имебщей на нем производные до $\large n$-го порядка включительно

Поэтому $\large z = \varphi(t)$ можно представить формулой Тейлора по степеням $\large t$
$$\Large \varphi(t) = \varphi(0) +\frac{\varphi'(0)}{1!}t+\frac{\varphi''(0)}{2!}t^2+\ldots+\frac{\varphi^{(n-1)}(0)}{(n-1)!}t^{n-1} + \frac{\varphi^{(n)}(\theta t)}{n!}t^n,~~0 < \theta < 1$$
Полагая $\large t = 1$, получим
$$\Large \varphi(1) = \varphi(0) + \frac{\varphi'(0)}{1!} +\frac{\varphi''(0)}{2!}+\ldots+\frac{\varphi^{(n-1)}(0)}{(n-1)!} + \frac{\varphi^{(n)}(\theta )}{n!},~~0 < \theta <1$$

Выразим величины правой части через исходную функцию $\large f(x,y)$ и ее производные

Аргументы $\large x,~y$ функции $\large f(x,y)$ являются функциями от $\large t$, но имеют постоянные дифференциалы $\large dx = \Delta x \cdot dt,~dy = \Delta y \cdot dt$, где $\large \Delta x,~\Delta y$ фиксированы. Поэтому для вычисления последовательных дифференциалов функции $\large z = f(x,y)$ применима формула
$$\Large d^pz = \bigg(\frac{\partial}{\partial x}dx+\frac{\partial z}{\partial y}dy\bigg)^p f(x,y) = $$
$$\Large =\bigg(\frac{\partial }{\partial x}(\Delta x~ dt) + \frac{\partial}{\partial y}(\Delta y ~dt)\bigg)^pf(x,y) = \bigg(\frac{\partial}{\partial x}\Delta x + \frac{\partial}{\partial y}\Delta y{}\bigg)^pf(x,y)(dt)^p$$
Откуда
$$\Large \frac{d^pz}{dt^p} = \varphi^{(p)}(t)=\bigg(\frac{\partial}{\partial x}\Delta x+\frac{\partial }{\partial y}\Delta y\bigg)^pf(x,y)$$

Для формулы Телйора кроме $\large \varphi^{(n)}(0)$ необходимо найти $\large \varphi(1),~\varphi^{(n)}(\theta)$

При $\large t = 0$ имеем $\large x = x_0,~y = y_0$
$$\Large \varphi^{(p)}(0) = \bigg(\frac{\partial}{\partial x}\Delta x+\frac{\partial }{\partial y}\Delta y\bigg)^pf(x,y)\textbar_{\begin{array}{}x=x_0\\y=y_0\end{array}}~~~~(p=  0,1,\ldots,n-1)$$

При $\large t = \theta$
$$\Large \varphi^{(n)}(\theta) = \bigg(\frac{\partial}{\partial x}\Delta x+\frac{\partial}{\partial y}\Delta y\bigg)^nf(x,y)\textbar_{\begin{array}{}x = x_0+\theta\Delta x\\y = y_0+\theta\Delta y\end{array}}$$

При $\large t = 1$
$$\Large \varphi(1) = f(x_0\Delta x, y_0+\Delta y)$$

Подставим полученные выражения в формулу Тейлора

$$\Large f(x_0+\Delta x, y_0 + \Delta y) = $$
$$\Large = f(x_0,y_0)+ \Big(\frac{\partial }{\partial x}\Delta x+\frac{\partial}{\partial y}\Delta y\Big)f(x,y)\textbar_{\begin{array}{}x = x_0\\y=y_0\end{array}} +$$
$$\Large +\frac{1}{2!}\Big(\frac{\partial }{\partial x}\Delta x+\frac{\partial}{\partial y}\Delta y\Big)^2f(x,y)\textbar_{\begin{array}{}x = x_0\\y=y_0\end{array}} +\ldots+$$
$$\Large +\frac{1}{(n-1)!}\Big(\frac{\partial }{\partial x}\Delta x+\frac{\partial}{\partial y}\Delta y\Big)^{n-1}f(x,y)\textbar_{\begin{array}{}x = x_0\\y=y_0\end{array}} +$$
$$\Large +\frac{1}{n!}\Big(\frac{\partial }{\partial x}\Delta x+\frac{\partial}{\partial y}\Delta y\Big)^nf(x,y)\textbar_{\begin{array}{}x = x_0+\Delta x\\y=y_0+\Delta y\end{array}},~~~0<\theta<1$$

Это формула Тейлора для функции $\large z = f(x,y)$, а 
$$\Large R_n = \frac1{n!}\Big(\frac{\partial }{\partial x}\Delta x+\frac{\partial}{\partial y}\Delta y\Big)^nf(x,y)\textbar_{\begin{array}{}x = x_0+\Delta x\\y=y_0+\Delta y\end{array}}$$
есть отстаточный член формулы в форме Лагранжа
