## Односторонние пределы функции в точке
### Левый предел функции
Пусть функция $\large f(x)$ определена на интервале $\large (a, x_0)$

Число $\large A$ называется **пределом функции $\large f(x)$ в точке $\large x_0$ слева**, если для любого $\large \varepsilon > 0$ существует такое $\large \delta>0$, что для всех $\large x$, удовлетворяющих условию
$\large x_0-\delta < x < x_0$, будет верно неравенство
$$\Large |f(x) - A| < \varepsilon$$

Обозначение $\Large A = \displaystyle\lim_{x\to x_0-0}f(x)$
### Правый предел функции
Пусть функция $\large f(x)$ определена на интервале $\large (x_0, b)$

Число $\large A$ называют **пределом функции $\large f(x)$ в точке $\large x_0$ справа**, если для любого $\large \varepsilon > 0$ существует такое $\large \delta > 0$, что для всех $\large x$, удовлетворяющих условию $\large x_0 < x < x_0+\delta$, будет верно неравенство
$$\Large |f(x)-A| < \varepsilon$$
Обозначение $\Large A = \displaystyle\lim_{x\to x_0+0}f(x)$

## Теорема о связи между односторонними пределами и пределом функции
Пусть функция $\large f(x)$ определена в проколотой окрестности точки $\large x_0$
>Для того, чтобы функция $\large f(x)$ имела предел в точке $\large x_0$, *необходимо и достаточно*, чтобы существовали пределы функции $\large f(x)$ в точке $\large x_0$ слева и справа и они были равны между собой. Тогда
>$$\Large f(x_0-0)=f(x_0+0)=\lim_{x\to x_0}f(x)$$

##### Доказательство
$\square$

###### Необходимость
Пусть $\large A = \displaystyle\lim_{x\to x_0-0} f(x) = \lim_{x\to x_0+0}f(x)$

Тогда для любого $\large \varepsilon > 0$ существуют такие $\large \delta_1>0,~\delta_2 > 0$, что если $\large x_0-\delta_1<x<x_0$ и соответственно $\large x_0 < x < x_0+\delta_2$, то $\large |f(x)-A| < \varepsilon$

Положим $\large \delta = \min\{\delta_1;\delta_2\}$

Тогда для всех $\large x : 0 < |x- x_0|<\delta$ получим
$$\Large |f(x)-A|<\varepsilon$$
Это означает, что 
$$\Large \lim_{x\to x_0}f(x) = A$$
###### Достаточность
Пусть $\large \displaystyle\lim_{x \to x_0}f(x) = A$. Тогда для каждого $\large \varepsilon > 0$ существует такое $\large \delta>0$, что для всех $\large x \in (x_0-\delta;~x_0+\delta), ~x\ne x_0$ верно неравенство 
$$\Large |f(x) - A| < \varepsilon$$

Т.к. неравенство имеет место на интервалах $\large (x_0-\delta,x_0)$ и $\large (x_0,x_0+\delta)$, то
$$\Large A = \lim_{x\to x_0-0}f(x)~~~~~~A = \lim_{x\to x_0+0}f(x)$$

$\blacksquare$
