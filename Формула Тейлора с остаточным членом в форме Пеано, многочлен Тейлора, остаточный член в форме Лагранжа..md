## Формула Тейлора
#### Формула Тейлора для многочлена степени $\Large n$

$$\Large P(x) = b_0+b_1x+b_2x^2+\ldots+b_nx^n$,~~b_0 = const \ne 0$$
 Для любого числа $\large a$ многочлен $\large P(x)$ можно представить в виде суммы степеней разности $\large (x-a)$, взятых с некоторыми коэффициентами
 
 Положим $\large x = a + t$
 
 Тогда
 $$\Large P(x) = P(a+t) = b_0 + b_1(a+t)+\ldots+b_n(a+t)^n$$
 Раскрывая в правой части скобки и группируя подобные члены, получим 
 $$\Large P(a+t) = A_0+A_1 t+A_2t^2+\ldots+A_nt^n$$
 
 Выразим $\large t$ обратно через $\large x$ ($\large t = x-a$)
 $$\Large P(x) = A_0+A_1(x-a)+A_2(x-a)^2+\ldots+A_n(x-a)^n$$
 где $\large A_0,A_1,\ldots,A_n$ - некоторые постоянные
 
 Продифференцируем многочлен $\large P(x)$ в новой форме $\large n$ раз по $\large x$
 $$\Large\begin{rcases}
 P'(x) = A_1+2A_2(x-a)+3A_3(x-a)^2+\ldots+nA_n(x-a)^{n-1}\\
 P''(x) = 2\cdot1A_2+3\cdot2A_3(x-a)+4\cdot3A_4(x-a)^2+\ldots+n(n-1)A_n(x-a)^{n-2}\\
 \ldots\ldots\ldots\\
 P^{(n)}(x) =n(n-1)(n-2)\ldots3\cdot2\cdot1A_n\cdot(x-a)^0
 \end{rcases}$$
 
 Положим в равенствах $\large x = a$
 $$\Large P(a) = A_0,~P'(a) = 1!A_1,~P''(a) = 2!A_2, P'''(a) = 3!A_2,~\dots,~P^{(n)}(a) = n!A_n$$
 Откуда
 $$\Large A_0 = P(a),~A_1 = \frac{P'(a)}{1!},~A_2 = \frac{P''(a)}{2!},~A_3 = \frac{P'''(a)}{3!},~\ldots,~A_n = \frac{P^{(n)}(a)}{n!}$$
 Следовательно многочлен $\large P(x)$ можно записать в виде
 >$$\Large P(x)= P(a)+\frac{P'(a)}{1!}(x-a)+\frac{P''(a)}{2!}(x-a)^2+\ldots+\frac{P^{(n)}(a)}{n!}(x-a)^n$$

Это и есть **формула Тейлора** по степеням $\large (x-a)$ для многчлена $\large P(x)$ степени $\large n$

## Многочлен Тейлора

Пусть теперь в окрестности точки $\large x = a$ задана функция $\large f(x)$, не являющаяся многочленом степени $\large n-1$, но имеющая в этой окрестности производные до $\large n$-го порядка включительно

Вычислим величины $\large f(a), ~f'(a),~f''(a),~\ldots~,f^{(n-1)}(a)$ и построим функцию
$$\Large Q_{n-1}(x) = f(a)+\frac{f'(a)}{1!}(x-a)+\ldots+\frac{f^{(n-1)}(a)}{(n-1)!}(x-a)^{n-1}$$
$\large Q_{n-1}(x)$ - многочлен степени $\large n-1$ называется **многочленом Тейлора** по степеням $\large (x-a)$ функции $\large f(x)$. Если бы функция $\large f(x)$ самая являлась многочленом степени $\large n-1$, то для всех $\large x$ из рассматриваемой окрестности выполнялось бы $\large f(x)\equiv Q_{n-1}(x)$, но т.к. $\large f(x)$ не является многочленом степени $\large n-1$, данное тождество не имеет места

В таком случае положим
$$\Large f(x) = Q_{n-1}(x)+R_n(x)$$
где $\large Q_{n-1}(x)$ - многочлен Тейлора степени $\large n-1$ функции $\large f(x)$ по степеням $\large (x-a)$, а $\large R_n(x)$ - **остаточный член** формулы Тейлора

$\large R_n(x)$ можно выразить через $\large n$-ю производную $\large f(x)$

Пусть функция $\large f(x)$ (не являющаяся многочленом степени $\large n-1$) на отрезке $\large [a,b]$ имеет непрерывные производные до $\large (n-1)$-го порядка включительно, а в интервале $\large (a,b)$ существует производная $\large f(x) ~n$-го порядка

$$\Large f(b) = f(a)+\frac{f'(a)}{1!}(b-a)+\frac{f''(a)}{2!}(b-a)^2+\ldots+\frac{f^{(n-1)}(a)}{(n-1)!}(b-a)^{n-1}+R_n$$

Будем искать $\large R_n$ в виде 
$$\Large R_n = M(b-a)^n$$
где величина $\large M$ подлежит определению

Введем вспомогательную функцию $\large \varphi(x)$, определив ее на отрезке $\large [a,b]$ равенством
$$\Large \varphi(x) = f(b) - \Big( f(x) + \frac{f'(x)}{1!}(b-x)+\ldots+\frac{f^{(n-1)}(x)}{(n-1)
!}(b-x)^{n-1}+M(b-x)^n\Big)$$

$\large \varphi(x)$ удовлетворяет всем условиям [[Теорема Ролля#Теорема Ролля о нуле производной|теоремы Ролля]]
- $\large \varphi(x)$ непрерывна на $\large [a,b]$, т.к. на $\large [a,b]$ непрерывна исходная функция $\large f(x)$ со всеми ее производными до $\large (n-1)$-го порядка включительно
- $\large \varphi(x)$ на интервале $\large (a,b)$ имеет производную, т.к. на $\large (a,b)$ имеет производную $\large n$-го порядка функция $\large f(x)$
- $\large \varphi(x)$ на концах отрезка $\large [a,b]$ имеет равные значения $\large \varphi(a) = 0 = \varphi(b)$

Поэтому существует такая точка $\large \xi\in(a,b)$, что $\large \varphi'(\xi) = 0$

Найдем эту производную
$$\Large \varphi'(x) = -\Big(f'(x)-\frac{f'(x)}{1!}+\frac{f''(x)}{1!}(b-x)-\frac{f''(x)}{2!}\cdot2\cdot(b-x)+\ldots$$
$$\Large \ldots+\frac{f^{(n-1)}(x)}{(n-2)!}(b-x)^{n-2}-\frac{f^{(n-1)}(x)}{(n-1)!}\cdot(n-1)\cdot(b-x)^{n-2}+\frac{f^{(n)}(x)}{(n-1)!}(b-x)^{n-1}+Mn(b-x)^{n-1}\Big)$$
все члены, кроме двух последних сокращаются, т.е.
$$\Large \varphi'(x) = -\frac{f^{(n)}(x)}{(n-1)!}(b-x)^{n-1}+Mn(b-x)^{n-1}$$ 
Таким образом 
$$\Large \varphi'(x) = -(b-\xi)^{n-1}\cdot\Big(\frac{f^{(n)}(\xi)}{(n-1)!}-Mn\Big) = 0$$
$\large \xi \in (a,b)\Rightarrow \xi \ne b\Rightarrow$
$$\Large\Rightarrow\frac{f^{(n)}(\xi)}{(n-1)!}-M\cdot n = 0$$
Получаем выражение для $\large R_n$
$$\Large R_n = \frac{f^{(n)}(\xi)}{n!}(b-a)^n,~~a<\xi<b$$
Подставив найденное $\large R_n$ в выражение $\large f(b)$, получим
$$\Large f(b) = f(a)+\frac{f'(a)}{1!}(b-a)+\ldots+\frac{f^{(n-1)}(a)}{(n-1)!}+\frac{f^{(n)}(\xi)}{n!}(b-a)^n$$
Эта формула называется **формулой Тейлора** для функции $\large f(x)$
Последнее слагаемое 
$$\Large R_n = \frac{f^{(n)}(\xi)}{n!}(b-a)^n$$
где $\large \xi$ - некоторая точка между $\large a$ и $\large b$, называется **остаточным членом  формулы Тейлора в форме Лагранжа**

В формуле Тейлора вместо $\large a$ и $\large b$ можно взять две любые точки $\large x$ и $\large x_0$ из отрезка $\large [a,b]$ и записать ее в виде
$$\Large f(x) = f(x_0)+\frac{f'(x_0)}{1!}+\ldots+\frac{f^{(n-1)}(x_0)}{(n-1)!}(x-x_0)^{n-1}+\frac{f^{(n)}(x_0+\theta(x-x_0))}{n!}(x-x_0)^n$$
где $\large \theta \in (0,1)$

## Формула Маклорена
**Формула Маклорена** - частный случай формулы Тейлора, где $\large a = 0$, т.е. разложение многочлена по формуле Тейлора по степеням $\large x$
$$\Large P(x) = P(0)+\frac{P'(0)}{1!}x+\frac{P''(0)}{2!}x^2+\ldots+\frac{P^{(n)}}{n!}x^n$$

Эта функция получается, если из $\large f(b)$ вычесть выражение $\large f(b)$, в котором точка $\large a$ заменена на $\large x$
