Пусть функция $\large \alpha(x)$ определена в некоторой проколотой окрестности точки $\large x_0$

Функция $\large \alpha(x)$ называется **бесконечно малой функцией при $\large x \to x_0$**, если 
$$\Large \lim_{x\to x_0}\alpha(x)=0$$

или же
> Функция $\large \alpha(x)$ - б.м.ф. при $\large x \to x_0$, если $\large \forall\varepsilon > 0~~\exists\delta>0 :\forall x,~ 0 < |x-x_0|<\delta \Rightarrow |\alpha(x)| < \varepsilon$ 

Функция $\large \alpha(x)$ - б.м.ф. при $\large x \to \infty$, если
$$\Large \lim_{x\to\infty}\alpha(x) = 0$$

При $\large x \to +\infty$, если
$$\Large \lim_{x\to+\infty}\alpha(x) = 0$$

При $\large x \to -\infty$, если
$$\Large \lim_{x\to-\infty}\alpha(x) = 0$$

#### Примеры
- $\Large\alpha(x) = x- x_0 \Rightarrow \displaystyle\lim_{x\to x_0}\alpha(x) = 0$
- $\Large\alpha(x) = \frac1x, ~x\ne0 \Rightarrow \displaystyle\lim_{x\to \infty}\alpha(x) = 0$
- $\Large\alpha(x) = e^{-x} \Rightarrow \displaystyle\lim_{x\to +\infty}\alpha(x) = 0$

## Свойства бесконечно малых функций
###### Доказать два любых
### Теорема о сумме б.м.ф.
> Если $\large \alpha(x)$ и $\large \beta(x)$ - б.м.ф. при $\large x\to x_0$, то их сумма $\large \alpha(x) + \beta(x)$ также является бесконечно малой при $\large x\to x_0$

##### Доказательство
$\square$

Возьмем любое $\large \varepsilon > 0$

Т.к. $\large \alpha(x)$ - б.м.ф. при $\large x\to x_0$, то найдется $\large \delta_1$ такое, что для всех $\large x$, удовлетворяющих условию
$$\Large 0 < |x-x_0| < \delta_1$$
будет верно неравенство 
$$\Large |\alpha(x)|<\frac\varepsilon2$$

Т.к. $\large \beta(x)$ - б.м.ф. при $\large x\to x_0$, то найдется $\large \delta_2$ такое, что для всех $\large x$, удовлетворяющих условию
$$\Large 0 < |x-x_0| < \delta_2$$
будет верно неравенство 
$$\Large |\beta(x)|<\frac\varepsilon2$$

Положим $\large \delta = \min\{\delta_1,\delta_2\}$ 

Тогда для всех $\large x : 0 <|x-x_0| <\delta$ будут одновременно верны оба неравенства, поэтому

$$\Large |\alpha(x)+\beta(x)| \le |\alpha(x)| + |\beta(x)| < \frac\varepsilon2 + \frac\varepsilon2 = \varepsilon~~\forall x: ~0<|x-x_0|<\delta$$

Это означает, что сумма $\large \alpha(x) + \beta(x)$ при $\large x\to x_0$ является бесконечно малой функцией

$\blacksquare$

*Теорема справедлива для любого конечного числа б.м.ф.*

### Теорема о произведении б.м.ф. на ограниченную функцию
> Если функция $\large \alpha(x)$ - б.м.ф. при $\large x\to x_0$, а функция $\large f(x)$ ограничена в окрестности точки $\large x_0$, то произведение $\large \alpha(x)\cdot f(x)$ - бесконечно малая функция при $\large x\to x_0$
##### Доказательство
$\square$

Т.к. $\large f(x)$ ограничена в окрестности точки $\large x_0$, то существуют числа $\large \delta_1 > 0$ и $\large M >0$ такие, что
$$\Large |f(x)|\le M ~~\forall x \in (x_0-\delta_1;~x_0+\delta_1)$$

Возьмем любое $\large \varepsilon > 0$

Т.к. $\large \alpha(x)$ - б.м.ф. при $\large x\to x_0$, то найдется такое $\large \delta_2$, что для всех $\large x : 0 < |x-x_0| <\delta_2$ будет верно неравенство
$$\Large |\alpha(x)| < \frac\varepsilon M$$

Положим $\large \delta = \min \{\delta_1,\delta_2\}$, тогда для всех $\large x : 0 < |x-x_0| < \delta$ будут одновременно верны неравенства 
$$\Large |f(x)|< M~~~\text{и}~~~ |\alpha(x)|<\frac\varepsilon M$$ 

Тогда
$$\Large |f(x)\cdot\alpha(x)| = |\alpha(x)|\cdot|f(x)| < \frac\varepsilon M\cdot M = \varepsilon,~~\forall x:0<|x-x_0|<\delta$$
Это означает, что произведение $\large \alpha(x)\cdot f(x)$ - б.м.ф. при $\large x\to x_0$

$\blacksquare$

##### Следствие:

Если $\large \alpha(x)$ - б.м.ф. при $\large x \to x_0$, а $\large f(x)$ в точке $\large x_0$ имеет конечный предел, то произведение $\large \alpha(x)\cdot f(x)$ - б.м.ф. при $\large x\to x_0$

#### Лемма 
>Если функция $\large f(x)$ в точке $\large x_0$ имеет предел, отличный от нуля, то функция $\large \frac1{f(x)}$ ограничена в проколотой окрестности точки $\large x_0$

##### Доказательство
$\square$

Пусть $\large \displaystyle\lim_{x\to x_0}f(x) = A \ne 0$

Тогда для любого $\large \varepsilon > 0$, в частности для $\large \varepsilon = \frac{|A|}2$, найдется такое число $\large \delta > 0$, что для всех $\large x : 0 < |x-x_0| < \delta$ будет верно неравенство
$$\Large |A-f(x)| < \frac{|A|}2$$
Т.к. $\large |A-f(x)| \ge |A| - |f(x)|$, то 
$$\Large |A|- |f(x)| < \frac{|A|}2$$
Откуда
$$\Large |f(x)| > \frac{|A|}2~~\forall x:0<|x-x_0|<\delta$$
 
 Значит для всех указанных значений $\large x$ функция $\large f(x) > 0 \Rightarrow \frac{1}{f(x)}$ определена, причем
 $$\Large \Bigg|\frac{1}{f(x)}\Bigg| < \frac2{|A|}$$
 
 Так что функция $\large f(x) =\frac1{f(x)}$ ограничена в проколотой окрестности точки $\large x_0$

$\blacksquare$

## Теорема о частном б.м.ф. и функции, имеющей отличный от нуля предел
>Если $\large \alpha(x)$ -  б.м.ф. при $\large x \to x_0$, а функция $\large f(x)$ имеет в точке $\large x_0$ предел, отличный от нуля, то частное $\large \frac{\alpha(x)}{f(x)}$ - б.м.ф. при $\large x\to x_0$
##### Доказательство
$\square$

Представим частное $\large \frac{\alpha(x)}{f(x)}$ в виде
$$\Large \frac{\alpha(x)}{f(x)} = \alpha(x)\cdot\frac1{f(x)}$$

В силу [[Бесконечно  малые  функции. Свойства б.м.ф.#Лемма|леммы]] функция $\large \frac1{f(x)}$ ограничена в проколотой окрестности точки $\large x_0$, следовательно функция $\large \alpha(x)\cdot\frac1{f(x)}$ - б.м.ф. при $\large x\to x_0$ как произведение б.м.ф. на ограниченную

$\blacksquare$
